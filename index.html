DGConnect

DGConnect is a QGIS Plugin that enables users with social media, text, vectors, raster search and analytic capabilities on various commercial and free to use data sources.

View on GitHub Download .zip

Extract

Extract the DGConnect folder and move it to C:\Users.qgis2\python\plugins

Install

Open QGIS and go to Manage and Install Plugins. Under Installed, check box next to DGX Plugin.

Settings

Click the settings icon. Once the username, password, and max items to return have been entered click Validate to ensure everything is working properly.
